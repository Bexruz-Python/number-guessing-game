# Number Guessing Game

A simple Python game where the computer thinks of a number between 1 and 10, and the player tries to guess it.

## How to play

1. Run the Python file: `number_guessing_game.py`
2. Enter your guess when prompted.
3. The game will tell you if your guess is too high, too low, or correct.
4. Keep guessing until you find the number.

## Requirements

- Python 3.x
